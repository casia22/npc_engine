## npc_engine文档维护
项目使用mkdocs来维护文档。
参考资料:
https://mkdocs-like-code.readthedocs.io/zh_CN/latest/get-started/create-program/

### 本地预览
```bash
mkdocs serve
```

### 项目结构

mkdocs.yml：此为配置文件，文档的结构、主题都可以在此设置。

docs文件夹：撰写的 Markdown 文档都放在这个文件夹内。

index.md：默认首页。

可以访问 https://docs.cognimatrix.games/npc_engine_doc/ 查看文档。
